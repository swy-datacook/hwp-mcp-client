# 🛠️ HWP MCP 도구 활용 가이드

## 📋 사용 가능한 도구 목록

### 1. 📄 문서 관리 (기본)

#### 문서 생성/열기/저장
```
✅ hwp_create() - 새 문서 생성
✅ hwp_open(path) - 기존 문서 열기
✅ hwp_save(path) - 문서 저장
✅ hwp_close_document(save) - 문서 닫기
✅ hwp_close_all_documents(save) - 모든 문서 닫기
```

**사용 예시**:
- "새 HWP 문서를 만들어주세요"
- "C:/문서/보고서.hwp 파일을 열어주세요"
- "현재 문서를 C:/문서/완성본.hwp로 저장해주세요"

---

### 2. 📝 텍스트 편집

#### 텍스트 입력 및 서식
```
✅ hwp_insert_text(text, preserve_linebreaks) - 텍스트 삽입
✅ hwp_set_font(name, size, bold, italic) - 글꼴 설정
✅ hwp_insert_paragraph() - 단락 추가
✅ hwp_get_text() - 문서 텍스트 가져오기
```

**사용 예시**:
- "제목에 '2026년 1월 주간 보고서'를 입력하고 크기 20, 굵게 설정해주세요"
- "본문에 다음 내용을 입력해주세요: [내용]"
- "현재 문서의 모든 텍스트를 보여주세요"

---

### 3. 📊 표 작업 (강력 추천!)

#### 표 생성 및 데이터 입력
```
✅ hwp_insert_table(rows, cols) - 표 생성
✅ hwp_create_table_with_data(rows, cols, data, has_header) - 데이터와 함께 표 생성
✅ hwp_fill_table_with_data(data, start_row, start_col, has_header) - 기존 표에 데이터 채우기
✅ hwp_fill_column_numbers(start, end, column, from_first_cell) - 열에 연속 숫자 채우기
✅ hwp_fill_cells(cells) - 특정 셀에 데이터 입력
✅ hwp_table_view(depth) - 표 구조 확인
✅ hwp_find_and_show_cell(text) - 특정 텍스트가 있는 셀 찾기
```

**사용 예시**:
- "5행 3열 표를 만들어주세요"
- "다음 데이터로 표를 만들어주세요: [['이름', '나이', '직급'], ['홍길동', '30', '대리'], ...]"
- "첫 번째 열에 1부터 10까지 숫자를 채워주세요"
- "표의 구조를 보여주세요"

✅ hwp_undo(count) - 실행 취소

## 💡 활용 팁

### 1. 표 작업이 가장 강력합니다!
- `hwp_create_table_with_data()` - 데이터와 함께 표를 한 번에 생성
- `hwp_fill_table_with_data()` - 기존 표에 데이터 채우기
- `hwp_table_view()` - 표 구조 확인 후 작업

### 2. 일괄 작업으로 효율성 UP
- `hwp_batch_operations()` - 여러 작업을 한 번에 실행
- 반복 작업을 자동화할 때 유용

### 3. 템플릿 활용
- 템플릿 파일: `C:\hwp-mcp\templates\weekly_report_template.hwp`
- 템플릿을 열고 → 찾기/바꾸기로 내용 수정 → 저장

### 4. 다중 문서 작업
- 여러 문서를 동시에 열고 탭으로 전환하며 작업 가능
- `hwp_list_tabs()` → `hwp_switch_tab()` 활용

### 5. 에러 발생 시
- `hwp_undo()` - 작업 취소
- `hwp_close_document(save=False)` - 저장하지 않고 닫기
- `hwp_ping_pong()` - 연결 테스트

## 📊 도구 우선순위 (추천)

### 🥇 가장 많이 사용할 도구
1. `hwp_create()` - 문서 생성
2. `hwp_insert_text()` - 텍스트 입력
3. `hwp_create_table_with_data()` - 표 생성 및 데이터 입력
4. `hwp_save()` - 문서 저장
5. `hwp_set_font()` - 글꼴 설정

### 🥈 자주 사용할 도구
6. `hwp_open()` - 문서 열기
7. `hwp_fill_table_with_data()` - 표 데이터 채우기
8. `hwp_replace_text()` - 찾기/바꾸기
9. `hwp_insert_paragraph()` - 단락 추가
10. `hwp_table_view()` - 표 구조 확인

### 🥉 고급 사용자용
11. `hwp_batch_operations()` - 일괄 작업
12. `hwp_create_complete_document()` - 완성 문서 생성
13. `hwp_list_tabs()` / `hwp_switch_tab()` - 다중 문서 관리
14. `hwp_fill_column_numbers()` - 자동 번호 매기기
